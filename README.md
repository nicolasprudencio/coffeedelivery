## 💿 Como rodar o projeto

```bash
# clone este repositório

$ git clone linkrepo

# Acesse a pasta do projeto no seu terminal

$ cd coffee-delivery

# Instale as dependências

$ npm install

# Execute a aplicação

$ npm run dev

# A Aplicação será iniciada na porta 3000
# http://localhost:3000

```
